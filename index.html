Amador Finance Hub

A personal, single-user financial management application operated by its owner for their own household finances. This application is not a commercial product, is not offered to the public, and has no users other than its owner.

Purpose
Personal budgeting, balance tracking, and transaction analysis for the owner's own bank accounts.
Users
One — the application owner. No third-party users, no sign-ups.
Data aggregator
Bank connectivity is provided by Plaid. Bank credentials are never seen or stored by this application; institutions using OAuth authenticate directly with the bank.
Hosting
Self-hosted on the owner's private infrastructure. Not exposed as a public service.

Data Access & Use

Through Plaid, this application accesses only the following data from the owner's linked accounts:

  • Account information — account names, types, and balances.
  • Transactions — transaction history for budgeting and expense analysis.

How this data is used:

  • Displayed to the owner in private dashboards for personal budgeting and analysis.
  • Stored on the owner's private, self-hosted infrastructure.

What is never done with this data:

  • No selling, sharing, or transferring data to third parties.
  • No advertising, profiling, or commercial use of any kind.
  • No data collection from anyone other than the owner.

Privacy Policy

Scope. This application has a single user — its owner. It does not collect personal information from visitors of this page or from any third party.

Data storage & security. Financial data retrieved via Plaid is stored on the owner's private infrastructure, protected by TLS encryption in transit, access controls, and routine encrypted backups. Access tokens are stored securely and never exposed publicly.

Data retention & deletion. Data is retained only as long as useful for personal financial analysis and can be deleted by the owner at any time. Bank connections can be revoked at any time through the linked institution or via Plaid Portal.

Third parties. The only third-party service processing financial data is Plaid, acting as data aggregator. See Plaid's End User Privacy Policy.

Contact

For any inquiry regarding data privacy or security practices of this application, contact the owner at: franciscoamadorr@gmail.com

Amador Finance Hub

Aplicación personal de gestión financiera, de un solo usuario, operada por su propietario para sus finanzas domésticas. No es un producto comercial, no se ofrece al público y no tiene usuarios distintos a su propietario.

Propósito
Presupuesto personal, seguimiento de saldos y análisis de transacciones de las cuentas bancarias del propietario.
Usuarios
Uno — el propietario de la aplicación. Sin terceros, sin registros.
Agregador de datos
La conectividad bancaria la provee Plaid. Esta aplicación nunca ve ni almacena credenciales bancarias; las instituciones con OAuth autentican directamente con el banco.
Alojamiento
Auto-hospedada en infraestructura privada del propietario. No expuesta como servicio público.

Acceso y uso de datos

A través de Plaid, esta aplicación accede únicamente a los siguientes datos de las cuentas vinculadas del propietario:

  • Información de cuentas — nombres, tipos y saldos.
  • Transacciones — historial de transacciones para presupuesto y análisis de gastos.

Los datos no se venden, comparten ni transfieren a terceros, no se usan con fines publicitarios ni comerciales, y no se recolecta información de ninguna persona distinta al propietario.

Política de privacidad

Almacenamiento y seguridad. Los datos financieros obtenidos vía Plaid se almacenan en infraestructura privada del propietario, protegidos con cifrado TLS en tránsito, controles de acceso y respaldos cifrados rutinarios.

Retención y eliminación. Los datos se conservan solo mientras sean útiles para el análisis financiero personal y pueden eliminarse en cualquier momento. Las conexiones bancarias pueden revocarse en cualquier momento desde la institución vinculada o vía Plaid Portal.

Contacto. Para consultas sobre privacidad o seguridad de datos: franciscoamadorr@gmail.com
